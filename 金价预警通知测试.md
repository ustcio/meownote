# 金价预警通知测试

## 测试命令

### 测试飞书通知
```bash
curl "https://api.agiera.net/api/gold/alert/test?type=feishu"
```

### 测试邮件通知
```bash
curl "https://api.agiera.net/api/gold/alert/test?type=email"
```

### 同时测试所有通知
```bash
curl "https://api.agiera.net/api/gold/alert/test?type=all"
```
