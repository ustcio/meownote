## 🔧 1. Swiper 核心算法

### 1.1 slideTo 滑动算法

```javascript
// 核心逻辑
slideTo(index, speed, runCallbacks) {
  // 1. 计算目标索引
  const realIndex = this.loop ? index % this.slides.length : index;
  
  // 2. 处理循环边界
  if (this.params.loop) {
    index = index + this.loopedSlides;
  }
  
  // 3. 计算位移距离
  const translate = -index * (this.slideSize + this.params.spaceBetween);
  
  // 4. 应用CSS transform
  this.setTranslate(translate);
  
  // 5. 触发过渡动画
  this.setTransition(speed);
  
  // 6. 更新状态
  this.updateActiveIndex(realIndex);
  this.updateSlidesClasses();
}
```

**算法要点:**
- 使用 `translate3d()` 实现硬件加速
- 通过 `requestAnimationFrame` 优化动画性能
- 支持缓动函数 (ease, ease-in, ease-out等)

### 1.2 Autoplay 自动播放算法

```javascript
class Autoplay {
  constructor() {
    this.timeout = null;
    this.startTime = Date.now();
    this.paused = false;
  }
  
  // 启动自动播放
  start() {
    this.stop();
    this.timeout = setTimeout(() => {
      this.swiper.slideNext();
      this.start(); // 递归调用
    }, this.params.delay);
  }
  
  // 暂停机制
  pause() {
    if (this.timeout) {
      clearTimeout(this.timeout);
      this.timeout = null;
    }
  }
  
  // 恢复播放
  resume() {
    if (!this.paused) {
      this.start();
    }
  }
}
```

**触发暂停的场景:**
- 用户触摸/鼠标悬停
- 页面不可见 (visibilitychange)
- 手动滑动后延迟恢复

### 1.3 Loop 循环算法

```javascript
// 循环实现原理
createLoop() {
  // 1. 克隆slides
  const slidesToClone = this.params.slidesPerView + this.params.loopAdditionalSlides;
  
  // 2. 克隆开头和结尾
  for (let i = 0; i < slidesToClone; i++) {
    // 克隆末尾到开头
    const cloneEnd = this.slides[this.slides.length - slidesToClone + i].cloneNode(true);
    this.wrapperEl.prepend(cloneEnd);
    
    // 克隆开头到末尾
    const cloneStart = this.slides[i].cloneNode(true);
    this.wrapperEl.append(cloneStart);
  }
  
  // 3. 调整初始位置
  this.setTranslate(-this.loopedSlides * this.slideSize);
}

// 循环切换时的索引重置
onTransitionEnd() {
  if (this.activeIndex <= this.loopedSlides) {
    // 跳到真实末尾
    this.slideTo(this.slides.length - this.loopedSlides * 2, 0, false);
  } else if (this.activeIndex >= this.slides.length - this.loopedSlides) {
    // 跳到真实开头
    this.slideTo(this.loopedSlides, 0, false);
  }
}
```

### 1.4 触摸滑动算法

```javascript
onTouchStart(event) {
  // 记录起始位置和时间
  this.touchStartX = event.touches[0].pageX;
  this.touchStartY = event.touches[0].pageY;
  this.touchStartTime = Date.now();
}

onTouchMove(event) {
  // 计算位移
  const currentX = event.touches[0].pageX;
  const diff = currentX - this.touchStartX;
  
  // 应用阻力效果
  const resistanceRatio = this.params.resistanceRatio;
  const translate = this.currentTranslate + diff * resistanceRatio;
  
  this.setTranslate(translate);
}

onTouchEnd(event) {
  // 计算滑动速度
  const timeDiff = Date.now() - this.touchStartTime;
  const distance = this.currentX - this.touchStartX;
  const velocity = distance / timeDiff;
  
  // 判断是否切换slide
  if (Math.abs(velocity) > this.params.swipeThreshold) {
    // 快速滑动 - 切换到下一张
    if (velocity > 0) {
      this.slidePrev();
    } else {
      this.slideNext();
    }
  } else if (Math.abs(distance) > this.slideSize * 0.5) {
    // 滑动距离超过50%
    if (distance > 0) {
      this.slidePrev();
    } else {
      this.slideNext();
    }
  } else {
    // 回弹到当前位置
    this.slideTo(this.activeIndex);
  }
}
```

## 📐 4. 数学算法详解

### 4.1 缓动函数 (Easing Functions)

```javascript
// ease-out 缓动
function easeOut(t) {
  return 1 - Math.pow(1 - t, 3);
}

// ease-in-out 缓动
function easeInOut(t) {
  return t < 0.5 
    ? 4 * t * t * t 
    : 1 - Math.pow(-2 * t + 2, 3) / 2;
}

// cubic-bezier 贝塞尔曲线
function cubicBezier(t, p0, p1, p2, p3) {
  const u = 1 - t;
  return (
    u*u*u*p0 + 
    3*u*u*t*p1 + 
    3*u*t*t*p2 + 
    t*t*t*p3
  );
}
```

## 📝 总结

Claude Cowork页面的轮播系统是一个**多层次架构**:

1. **基础层**: Webflow Slider 提供核心滑动功能
2. **增强层**: Swiper 11 提供高级特性(循环、自动播放等)
3. **动画层**: GSAP 提供滚动触发和文字动画
4. **交互层**: Lottie 提供微交互动画

**核心算法**:
- 使用 `translate3d()` 实现硬件加速滑动
- 通过克隆slides实现无缝循环
- 使用定时器+递归实现自动播放
- 基于速度和距离判断滑动方向
- 应用缓动函数实现平滑动画
